📋 1. Présentation

Kitview Search Démo est une interface simplifiée destinée à la démonstration du moteur de recherche orthodontique Kitview Search V5.2.

Elle permet de :

  • Faire défiler automatiquement des exemples de recherches multilingues
  • Effectuer des recherches manuelles en texte libre ou vocalement
  • Présenter les fiches patients avec pathologies, tags et scores de similarité
  • Démontrer la fonctionnalité « Même X que... » (recherche par similarité)
💡 Cette page est conçue pour les salons, présentations et formations. Pour l'utilisation complète, voir l'interface standard (chat30.html).

🖥️ 2. Interface

Le bandeau supérieur contient :

2.2 Zone de configuration (écran d'accueil)

Au centre de l'écran d'accueil, la zone de configuration démo propose :

  • Sélecteur de démo : choix parmi les démos prédéfinies
  • Textarea : liste d'exemples de recherche (un par ligne). Les lignes commençant par # sont ignorées
  • Bouton ▶ Lancer : démarre le cycle automatique
  • Bouton ■ Stop : arrête la démo (apparaît pendant l'exécution)
  • Bouton ? Aide : ouvre cette page

2.3 Zone de recherche

La barre de recherche en bas de l'écran contient :

  • Champ de saisie texte libre (multilingue)
  • Bouton microphone 🎤 (recherche vocale)
  • Bouton envoi ⬆️

🎬 3. Mode démo

3.1 Configuration

La zone textarea accepte une liste d'exemples de recherche, un par ligne :

# Exemples français patients qui grincent des dents Patientes avec béance antérieure et classe II # Exemples multilingues 歯ぎしり Number of patients with linguo-version? Patientinnen mit Frontzahnlücke und Klasse II
💡 Les lignes vides et les lignes commençant par # sont ignorées. Vous pouvez ainsi organiser vos exemples par thème ou par langue.

3.2 Cycle automatique

Lorsque la démo est lancée :

  1. Phase 1 (10%) : Réinitialisation de l'écran, retour à l'accueil
  2. Phase 2 (20%) : Sélection aléatoire d'un exemple, affichage dans la barre de recherche
  3. Phase 3 (70%) : Lancement de la recherche, affichage des résultats
  4. Puis le cycle recommence — avec 1 chance sur 3 de réinitialiser l'écran

La durée totale d'un cycle est configurable dans les paramètres web (demoDuration). L'anneau de progression autour du switch Démo indique visuellement l'avancement.

⚠️ Pendant la démo automatique, la barre de recherche manuelle reste active. Toute recherche manuelle s'intègre au flux d'affichage.

🔍 4. Recherche manuelle

4.1 Recherche texte

Tapez votre question en langage naturel dans la barre en bas de l'écran. Le moteur comprend le français et 13 autres langues.

Types de recherche supportés :

  • Pathologies : « patients avec bruxisme et béance »
  • Démographie : « patientes de moins de 30 ans »
  • Combiné : « ado diabétique avec problème de bruxisme »
  • Comptage : « combien de patients avec classe II ? »
  • Angles : « patients avec un angle ANB = 0° »

4.2 Recherche vocale 🎤

Cliquez sur le microphone et parlez. La reconnaissance vocale transcrit votre question puis lance automatiquement la recherche.

💡 La langue de reconnaissance vocale suit la langue sélectionnée dans l'interface.

📊 5. Résultats

Les résultats s'affichent en mode chat (scroll vers le bas, style conversationnel) :

  • La question apparaît en haut de chaque bloc
  • Un bandeau résumé indique le nombre de patients trouvés
  • Les fiches patients défilent en dessous
  • Un bouton Page suivante → permet de charger plus de résultats

5.1 Fiche patient

Chaque fiche contient :

ZoneContenu
PhotoPortrait du patient (ou initiales si absent)
IdentitéPrénom, nom, sexe (♂/♀), âge
ScoreBadge coloré de similarité (si photofit actif) : 🎯100%, 🟢≥80%, 🔵≥60%, 🟠<60%
PathologiesGroupées par tag, avec tags en sombre et détails en clair
IDNuméro patient en haut à droite

Cliquez sur une fiche pour la déplier et voir les détails complets.

5.2 Recherche par similarité — « Même X que... »

Chaque élément cliquable d'une fiche patient permet de lancer une recherche « Même X que [patient] » :

Élément cliquéRecherche déclenchée
Photo / portraitMême portrait (Photofit par similarité faciale)
Sexe (♂/♀)Même sexe
ÂgeMême âge (±3 ans)
Tag pathologieMême tag (ex: même « occlusion »)
Pathologie complèteMême pathologie exacte
💡 Les critères actifs apparaissent en rouge. Le patient de référence est mis en évidence avec un fond jaune stabilo.

5.3 Photofit — Recherche par portrait 📸

En cliquant sur la photo d'un patient, le système compare son portrait à ceux de tous les patients de la base et retourne les plus ressemblants.

Les scores de similarité sont affichés sous forme de badges colorés :

  • 🎯 100% Portrait de référence
  • 🟢 ≥80% Très similaire
  • 🔵 ≥60% Similaire
  • 🟠 <60% Peu similaire

🌐 6. Langue

Kitview Search détecte automatiquement la langue de la question et adapte la réponse.

14 langues supportées : français, anglais, allemand, espagnol, italien, portugais, polonais, roumain, thaï, arabe, chinois, japonais, coréen, russe.

Sélection manuelle

Cliquez sur le bouton 🌐 dans le header pour ouvrir le popup de langues. Chaque langue est représentée par un chip avec son drapeau. Le clic change immédiatement la langue de l'interface.

Option →fr

La case à cocher →fr force les réponses en français même si la question est posée dans une autre langue. Utile pour les présentations devant un public francophone.

🔍 8. Recherche dans le guide

🆕 La barre de recherche 🔍 « Chercher... » dans le bandeau permet de trouver n'importe quel mot ou expression dans ce mode d'emploi.

Tapez au moins 2 caractères. Les occurrences sont surlignées en jaune, l'occurrence courante en orange. Naviguez avec ◀ ▶ ou F3 / Shift+F3.

🤖 9. Assistant IA

🆕 Un assistant IA est disponible via le bouton bleu 💬 en bas à droite. Il répond à vos questions sur ce guide en utilisant le contenu de la page comme contexte.

L'assistant utilise un modèle IA (Claude ou GPT) hébergé côté serveur via des variables d'environnement sécurisées (ANTHROPIC_API_KEY / OPENAI_API_KEY). Aucune clé API n'est stockée côté navigateur.

Des suggestions de questions sont proposées au premier lancement du chat. L'historique de conversation est conservé pendant la session.
