GEN1964 v2 - Détection intelligente du sexe
============================================

Ce script utilise la base officielle INSEE des prénoms.

1️⃣ Téléchargez la base INSEE :
https://www.data.gouv.fr/fr/datasets/fichier-des-prenoms/

2️⃣ Renommez le fichier en :
prenoms_insee.csv

3️⃣ Placez-le dans le même dossier que :
- gen1964_v2.py
- DirCcxphotothumbsbase_yu.txt

4️⃣ Lancez :
python gen1964_v2.py

La détection du sexe sera basée sur les statistiques cumulées depuis 1900.

Plus le prénom est fréquent, plus la prédiction est fiable.
