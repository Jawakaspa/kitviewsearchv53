🌐 Comment traduire le guide

Utilisez la traduction automatique de votre navigateur pour lire le guide dans votre langue.

💡 Le guide est rédigé en français Tous les navigateurs modernes proposent une traduction automatique intégrée. Cette page vous explique comment l'activer selon votre navigateur.

Google Chrome

Méthode 1 : Clic droit

  1. Faites un clic droit n'importe où sur la page
  2. Sélectionnez "Traduire en [votre langue]"
  3. La page sera automatiquement traduite

Méthode 2 : Barre d'adresse

  1. Cliquez sur l'icône de traduction 🌐 dans la barre d'adresse (à droite)
  2. Choisissez votre langue dans le menu déroulant
  3. Cliquez sur "Traduire"

Méthode 3 : Menu

  1. Cliquez sur les trois points verticaux (⋮) en haut à droite
  2. Sélectionnez "Traduire"

Microsoft Edge

Méthode 1 : Clic droit

  1. Faites un clic droit n'importe où sur la page
  2. Sélectionnez "Traduire en [votre langue]"

Méthode 2 : Barre d'adresse

  1. Cliquez sur l'icône de traduction dans la barre d'adresse
  2. Sélectionnez la langue souhaitée
  3. Cliquez sur "Traduire"

Méthode 3 : Menu

  1. Cliquez sur les trois points horizontaux (⋯) en haut à droite
  2. Allez dans "Plus d'outils"
  3. Sélectionnez "Traduire la page"

Mozilla Firefox

⚠️ Firefox n'a pas de traduction intégrée par défaut Vous devez installer une extension comme "Google Translate" ou "To Google Translate" depuis le magasin d'extensions Firefox.

Avec l'extension Google Translate :

  1. Installez l'extension depuis addons.mozilla.org
  2. Faites un clic droit sur la page
  3. Sélectionnez "Traduire cette page"

Safari (Mac / iPhone / iPad)

Sur Mac :

  1. Cliquez sur l'icône de traduction (aA) dans la barre d'adresse
  2. Sélectionnez "Traduire en [votre langue]"

Sur iPhone / iPad :

  1. Appuyez sur le bouton aA dans la barre d'adresse
  2. Sélectionnez "Traduire en [votre langue]"

🔄 Alternative universelle : Google Translate

  1. Allez sur translate.google.com
  2. Cliquez sur l'onglet "Sites Web"
  3. Collez l'adresse de la page du guide
  4. Sélectionnez votre langue cible
  5. Cliquez sur le lien généré
✅ Astuce Une fois traduite, la page reste en mémoire. Si vous naviguez dans le guide, les nouvelles sections seront également traduites automatiquement.
