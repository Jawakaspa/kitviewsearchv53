🔍 La recherche dans le manuel

La barre de recherche en haut de l'écran vous permet de trouver instantanément n'importe quelle information dans le manuel.

Comment rechercher ?

  1. Cliquez dans la barre de recherche ou appuyez sur Ctrl + F
  2. Tapez votre recherche (ex: "importer photos")
  3. Les résultats apparaissent instantanément, surlignés en jaune
  4. Utilisez les boutons ◀ Préc. et Suiv. ▶ pour naviguer entre les résultats
💡 Astuce : Utilisez le filtre de section pour limiter la recherche à un chapitre spécifique du manuel.

Raccourcis clavier pour la recherche

Raccourci Action
F3 Résultat suivant
Shift + F3 Résultat précédent
Échap Fermer la recherche

🤖 L'assistant IA

L'assistant IA est votre guide personnel pour naviguer dans le manuel. Posez-lui vos questions en langage naturel !

🤖

Assistant Kitview

Bonjour ! Je suis là pour vous aider à utiliser Kitview. Posez-moi vos questions, par exemple :

Comment importer des photos ? Créer un nouveau patient Exporter en PDF

Comment utiliser l'assistant ?

  1. Cliquez sur le bouton 💬 en bas à droite de l'écran
  2. Tapez votre question dans la zone de texte
  3. L'assistant analyse le manuel et vous répond avec des liens vers les sections pertinentes
  4. Cliquez sur les liens pour accéder directement à la documentation
💡 Conseil L'assistant répond dans la langue du manuel. Si vous consultez la version anglaise, posez vos questions en anglais pour de meilleurs résultats.

Exemples de questions

  • "Comment créer un nouveau patient dans Kitview ?"
  • "Quelle est la différence entre les viewers ?"
  • "Comment exporter des photos vers mon ordinateur ?"
  • "Comment personnaliser les couleurs de l'interface ?"

📚 Les versions du manuel

Le manuel existe en 4 versions pour s'adapter à votre niveau d'expertise. Utilisez les chips colorés en haut de l'écran pour basculer d'une version à l'autre.

📘

Standard

Version complète avec toutes les informations

🟢

Débutant

Explications détaillées, analogies simples, pas à pas

🟡

Intermédiaire

Pour les utilisateurs ayant les bases

🔴

Expert

Essentiel uniquement, pour les utilisateurs avancés

💡 Conseil : Commencez par la version Standard. Si vous trouvez les explications trop techniques, passez à la version Débutant. Si vous maîtrisez déjà Kitview, la version Expert vous fera gagner du temps.

🌍 Les langues disponibles

Le manuel est disponible nativement en français et en anglais. Ces traductions ont été réalisées avec soin pour garantir la qualité du contenu.

Langues natives

  • 🇫🇷 Français - Version originale
  • 🇬🇧 Anglais - Traduction professionnelle

Traduction automatique

Sur le serveur web, d'autres langues sont disponibles via Google Translate : allemand, espagnol, italien, portugais, polonais, roumain, thaï, arabe et chinois.

⚠️ Note La traduction automatique peut parfois produire des résultats approximatifs. En cas de doute, référez-vous à la version française ou anglaise.
